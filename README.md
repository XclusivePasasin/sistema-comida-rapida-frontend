# sistema-comida-rapida-frontend

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Install Axios
```
npm install axios
```
### Install Tailwind CSS
```
1 - npm install -D tailwindcss postcss autoprefixer
2 - npx tailwindcss init -p

```
